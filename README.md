MoegirliOS
==========

###1.3版 － Develop 线
摇动随机词条，边栏显示，错误报告优化

此版本预计于2014年8月上旬提交App Store


###1.2版 － Master 线
细节上的修改
- 缓存处理
- 增加对GoogleAnalytic 的支持
- 向前向后时，记住上次阅读到的位置

此版本已经于2014年7月22日提交App Store


###1.1版
较上一个版本有极大的更改，最重要的是不含有自制的渲染引擎，而是简单的拉取网页，将不合适的元素进行删除，然后对界面进行微调。
- 全新的渲染方式,完美体现页面元素
- 大幅提升对表格的渲染能力
- 完善的外链处理
- 部分视频支持在线观看
- 横屏支持 

此版本已经于2014年7月15日提交App Store
